# iredis

iredis地址：[https://iredis.io/](https://iredis.io/)

iredis是交互有好的redis命令行工具，提供了与shell其他命令的交互能力。

## 安装方法

```shell
$ pip install iredis
```

或者

```
wget  https://github.com/laixintao/iredis/releases/latest/download/iredis.tar.gz  && tar -xzf iredis.tar.gz  && ./iredis
```

## 使用例子

```shell
get json |jq
```
